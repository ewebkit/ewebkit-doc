Detailed Description

The Ewk cookies API.

Note
If the libsoup network backend is being used the functions here will only work with the default SoupSession, which can be retrieved with ewk_network_defaul_soup_session_get(). If a different SoupSession is used and associated with a view with ewk_view_soup_session_set(), all cookie management will have to be done manually.
