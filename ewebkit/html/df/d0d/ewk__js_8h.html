EAPI Ewk_JS_Object* ewk_js_object_new ( const Ewk_JS_Class_Meta *  meta_cls)

Create a Ewk_JS_Object to be used in ewk_view_js_object_add. The Meta class's methods and properties are not modified but references to it are kept as long as the object created from it lives. All properties created here will be added to the object hash of properties. Properties using default_prop's get/set/del methods should also be added to the objects hash(see: ewk_js_object_properties_get). Methods must free the arguments they receive(see: ewk_js_variang_array_free).

Parameters
clsEwk_JS_Class that describes the object to be created.
Returns
The Ewk_JS_Object created.
